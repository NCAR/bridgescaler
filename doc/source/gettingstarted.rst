Getting Started
===============

Dependencies
------------
* scikit-learn
* numpy
* pandas
* xarray
* pydigest

Installation
------------
For a stable version of bridgescaler, you can install from PyPI.

.. code-block:: bash

    pip install bridgescaler

For the latest version of bridgescaler, install from github.

.. code-block:: bash

    git clone https://github.com/NCAR/bridgescaler.git
    cd bridgescaler
    pip install .
